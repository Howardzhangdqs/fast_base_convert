Base Converter

Convert numbers between different bases using optimized algorithms.

📝 How to enter numbers:

  • Bases 2-10: Use digits 0-9 (e.g., 1010 for binary)
  • Bases 11-36: Use 0-9, A-Z (e.g., FF for hex, Z for base-36)
  • Bases 37-62: Use 0-9, a-z, A-Z (e.g., aZ for mixed case)
  • With prefixes: xFF, 0b1010, 0o777 for hex/binary/octal
